# Ecommerce mobile
## Sobre o Projeto:
O projeto tem como objetivo:
- Registrar usuário com firebase
- Logar usuário
- Exibir pagina inicial
- Exibir lista de produtos
- Exibir detalhes do produto
- Adicionar produto ao carrinho
- Fazer pedido
### Frontend
- React native
- Tailwindcss
### Backend
- Firebase
   - Authentication
   - Firestore
   - Storage
